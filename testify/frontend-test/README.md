# Testify Frontend (Vite + React)

Starter project scaffolded for the Testify backend.

## Scripts

- `npm install`
- `npm run dev` -> starts Vite dev server
- `npm run build` -> build for production
- `npm run preview` -> preview build

## Notes

- API base is `http://localhost:5000/api` in `src/api/axios.js`
- Uses localStorage for JWT token storage
- Socket.io client connects to `http://localhost:5000`
